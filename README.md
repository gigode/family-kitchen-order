# 老吃家饭庄点菜单

一个无后端的老吃家饭庄点菜网站：菜品图片、分类筛选、搜索、加减份数、点单备注和一键复制。

## 使用

直接打开 `index.html` 即可浏览。也可以用任意静态服务器运行：

```powershell
python -m http.server 5173
```

## 图片来源

菜品图保存在 `assets/images/`，`image-data.js` 中保存本地图片路径和来源说明。已有自家实拍图的菜品优先使用 `C:\works\menuphoto` 中按文件名匹配的照片；其余图片，尤其是饮料酒水，按更具体的关键词从 [百度图片搜索](https://image.baidu.com/) 重新下载。后续替换时，只要覆盖对应的本地图片文件即可。
